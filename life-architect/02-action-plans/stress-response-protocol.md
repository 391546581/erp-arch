# 🚨 压力与崩溃应对协议 (Stress Response Protocol)

> **当系统主频过高、红灯闪烁或即将宕机时，请执行此应急指南。**

## 第一步：生理冷启动 (Stabilize)
*当处于极度焦虑或崩溃边缘时，大脑已被杏仁核劫持，不要尝试做任何复杂决策。*

1.  **断开输入**：关掉社交软件，离开让你不舒服的物理环境。
2.  **强制降温**：冷水洗脸或洗一个冷水澡（触发哺乳动物潜水反射，强制心率下降）。
3.  **深长呼吸**：执行 4-7-8 呼吸法（吸气4秒，憋气7秒，呼气8秒）循环5次。
4.  **感官还原**：观察并命名周围的 5 种颜色、4 种声音、3 种触感。

## 第二步：事实脱敏 (Clarify)
*将“感觉”从“事实”中剥离出来。*

1.  **倾倒笔记**：在白纸（或此文件下）无差别写下脑子里所有的噪音，不做任何修饰。
2.  **风险终点线**：
    *   这件事最坏的结果是什么？（具体到数字、生活受到的影响）
    *   这个最坏结果会毁灭我的“生命轴心”吗？（通常不会）
3.  **圈子分析**：
    *   哪些是我完全无法控制的？（**停止关注**）
    *   哪些是我可以微弱影响的？（**暂缓**）
    *   哪些是我现在就能立刻动作的？（**标记出来**）

## 第三步：最小化执行 (Backload)
*不要追求解决问题，只追求恢复控制感。*

1.  **设定“停工线”**：告诉自己，“今天我只需要完成 [这一个动作] 就可以去睡觉了”。
2.  **2 分钟原则**：只做那份“可能导致问题解决”的最小化开头動作（例如：仅仅打开邮件，但不回）。
3.  **寻找锚点**：回到你的 [生命愿景](../00-vision/life-vision.md)，告诉自己：“这只是长途旅行中的一个暗礁，不影响航向。”
