# 🧭 Life Architect — 个人能力矩阵与人生架构系统

> **从程序员到全栈人生玩家 — 一个可迭代的个人进化操作系统**

## 这是什么？

Life Architect 是一个以 **Markdown 文件** 为载体的个人能力评估与成长迭代系统。它不是空洞的鸡汤，而是一个**可量化、可追踪、可迭代**的人生操作系统。

## 核心理念

```
你的人生就是一家创业公司，你是唯一的 CEO。
技能是"术"，心性是"道"，二者缺一不可。
```

### 五大杠杆

| 杠杆 | 说明 | 核心问题 |
|------|------|---------|
| 💰 资本杠杆 | 让钱为你工作 | 你的被动收入能覆盖支出吗？ |
| 🤝 人脉杠杆 | 让人为你背书 | 你能一个电话解决什么问题？ |
| ⚡ 技术杠杆 | 让技术为你赚钱 | 你的代码/工具在帮多少人？ |
| 🧠 认知杠杆 | 让认知带来信息差 | 你比别人早知道什么？ |
| 🧘 心性杠杆 | 让内心成为基石 | 你能扛住多大的压力和诱惑？ |

## 核心架构：体 / 术 / 智 / 道

Life Architect 将人生能力拆解为四个层次，形成一个完整的演化闭环：

1.  **体 (Foundation)**：**身体健康**。一切能力的物理基座，决定了精力的上限。
2.  **术 (Skills)**：**六大实战能力**（投资、英语移民、社交、亲密关系、管理、创业、技术杠杆）。解决"如何获得财富与影响"的问题。
3.  **智 (Mindset)**：**底层心智模型**。思维操作系统，决定了学习效率和决策质量。
4.  **道 (Character)**：**心性修炼**。灵魂内核，决定了人生的方向、韧性和最终高度。

## 如何使用

### 1. 首次使用 (极简起步 🚀)
1. **害怕复杂？** 请直接阅读 [🚀 极简启动指南](00-vision/starter-guide.md)
2. **只想坚持习惯？** 直接去 [⚛️ 微习惯追踪器](02-action-plans/atomic-habits.md)
3. **准备深度规划？**
   - 阅读 `00-vision/life-vision.md`，写下你的人生愿景
   - 打开 `01-competency-matrix/assessment-template.md`，做第一次自我评估

### 2. 日常运作
- **每周**：检查 `02-action-plans/weekly-habits.md`，追踪习惯执行
- **每月**：更新 `02-action-plans/monthly-focus.md`，调整聚焦方向
- **每季度**：用 `03-iteration/iteration-guide.md` 做大复盘，更新能力评分

### 3. 迭代进化
- 在 `03-iteration/reviews/` 下创建季度复盘文件
- 在 `03-iteration/evolution-log.md` 记录关键突破
- 定期更新能力矩阵评分，观察成长轨迹

- **诚实评估** — 自欺欺人是最大的浪费
